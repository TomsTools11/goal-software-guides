# Notion New User Presentation Guide

## Purpose

This presentation guide is designed to onboard new GOAL team members to the **GOAL Knowledge Base** in Notion. It covers workspace navigation, key databases, daily workflows, and best practices so every team member can find what they need quickly and contribute effectively.

---

## Outline

### 1. Welcome and Overview

- What is the GOAL Knowledge Base?
- Why we use Notion as our central hub
- What you will learn in this session

### 2. Getting Started with Notion

- Logging in and accessing the workspace
- Quick tour of the Notion interface (sidebar, pages, search)
- Setting up your profile and notification preferences

### 3. Navigating the Knowledge Base Home Page

- Overview of the home page layout
- Understanding the two main sections: Notes/Notebooks/Resources and Training/Projects/Tasks
- Using the navigation buttons to move between areas
- The My Items page and how it filters content relevant to you
- Using the home icon buttons to return to the home page

### 4. Core Databases and How to Use Them

- **Notes** - Capturing and organizing your notes
- **Notebooks** - Grouping notes and resources by topic
- **Resources** - Finding SOPs, training materials, research, and web clips
- **Training/Learning** - Accessing lessons and learning paths
- **Projects** - Tracking active projects and deliverables
- **Tasks** - Managing your to-dos and action items

### 5. Working with Database Views

- What are views (Table, Gallery, Chart, etc.)
- Switching between views
- Using filters and sorts to find what you need
- Understanding grouped views

### 6. Creating and Editing Pages

- How to create a new page in a database
- Using templates for consistent formatting
- Setting properties (Type, Notebook, Topic, Project, etc.)
- Adding content with text, headings, callouts, toggles, and attachments

### 7. Search and Discovery

- Using Notion search to find pages and information
- Browsing by Notebook or Topic
- Using the Notes Inbox and Recently Added Resources sections

### 8. Collaboration Best Practices

- Commenting and discussions on pages
- Mentioning teammates with @
- Keeping pages organized with naming conventions and tagging
- When to create a new page vs. update an existing one

### 9. Using the KB Assistant

- What is the KB Assistant
- How to ask it questions about GOAL processes and knowledge
- Tips for getting the best answers

### 10. Tips, Tricks, and FAQs

- Keyboard shortcuts to save time
- Favoriting pages for quick access
- Common mistakes to avoid
- Where to go for help
